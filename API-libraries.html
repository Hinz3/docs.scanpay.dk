API Libraries

Scanpay has official API client libraries for different programming languages. Client libraries make it easier for you to integrate our APIs in your project. All client libraries follow the same scheme and provide the following three methods:

  • newURL: Create a new payment link (API)
  • seq: Make a sequence request (API)
  • handlePing: Verify and parse ping requests (API)
